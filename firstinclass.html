HTML

HTML stands for Hyper Text Markup Language. This is a tag based language where tags are created with greater than and less than signs.

For Example

<p>is a paragraph</p>

There are special characters, such as &lt; which would be a less than sign.

                
def fun(coding):
    print(coding, "is fun")

fun("HTML")

Inline vs Block

Inline means that it is inline with the text. Block level forces a new line.

Block Level: div, p, h1, h2, h3, h4, h5, h6, li

Inline elements are span, img, a, strong, em
